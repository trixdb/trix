# Trix CLI

Command line interface for TrixDB.

## Installation

### Linux / macOS

```bash
curl -fsSL https://install.trixdb.com | sh
```

### macOS (Homebrew)

```bash
brew install trixdb/tap/trix
```

### Windows (Scoop)

```powershell
scoop bucket add trixdb https://github.com/trixdb/scoop-trix
scoop install trix
```

## Usage

```bash
trix --help
```
